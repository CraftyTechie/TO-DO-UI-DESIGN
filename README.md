#  Workflow

A clean and minimal productivity app UI designed in Figma to help users organize and manage daily tasks efficiently.



##  Overview

**Workflow** is a simple task management UI concept focused on clarity, ease of use, and modern design principles. It is designed to help users stay productive by keeping their tasks organized in one place.


##  Screens

This project includes two main screens:

###  Home Screen
-Welcome screen
###  Tasks Screen
- Full list of tasks
- Ability to view and manage to-dos
- Clean and structured layout for productivity

##  Design Focus

- Minimal and clean UI
- Clear visual hierarchy
- Consistent spacing and alignment
- Modern card-based layout
- User-friendly experience
## Tools Used

- Figma (UI/UX Design)


##  Purpose

This project was created to practice UI design skills and explore how a simple productivity app can be structured for real-world use.
